При создании меню + подменю обычно делается список ссылок. Каждый элемент списка помимо ссылки содержит вложенный список ссылок - подменю (не в ссылке, а в li)

Каждое подменю скрывается по умолчанию, и показывается только при hover на элемент родительского меню, который его содержит. Позиционируется он тоже относительно родительского элемента меню.
